\# Project Requirements



\## Functional Requirements



1\. The system must contain at least 100 artworks  

2\. Users can register and log in  

3\. Search functionality by title, artist, and style is implemented  

4\. The system provides personalized recommendations  

5\. Users can create personal collections  



\## Non-Functional Requirements



1\. Response time < 2 seconds  

2\. Responsive design (desktop, tablet, mobile)  

3\. Security: password encryption, protection against SQL injection  

4\. Scalability: support for up to 1000 users
