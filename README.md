# Art Gallery AI Curator

A virtual art gallery with AI-powered artwork recommendations

## Project Description

An intelligent system for personalized artwork selection, creation of virtual exhibitions, and educational content about art.

## Key Features

- Artwork catalog  
- Personalized recommendations  
- Virtual exhibitions  
- Educational content about artists and art styles  

## Technologies

- **Frontend:** React, TypeScript  
- **Backend:** Node.js / Python  
- **Database:** PostgreSQL / MongoDB  
- **AI / ML:** TensorFlow, scikit-learn
